# QuickHire Portal

A modern MERN-based recruitment platform.

## Features
- User Authentication (JWT)
- Post and apply to jobs
- Admin and recruiter roles
- Clean UI with Navy & Yellow theme

## Setup

### Backend
```bash
cd server
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```
